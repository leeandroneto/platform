Docs reorganized on 2026-04-21. New structure:

- `docs/core/` — living docs (architecture, schema, design-reference, decisions, copy-positioning, integrations)
- `docs/plano/` — master launch plan + findings + `execucao/` phase docs
- `docs/produto/` — research per feature area (relatorio, onboarding, checkout, landing). Create folder when phase starts, not before.
- `docs/archive/` — old docs, never read for execution

**How to apply:** Each new conversation starts with `"Leia docs/plano/PLANO_LANCAMENTO.md — nessa conversa trataremos da Fase X"`. See `docs/README.md` for full index.
