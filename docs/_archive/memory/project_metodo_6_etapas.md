7 etapas do método onboarding.bio, cada uma com produto vinculado:

1. **Captação** → Formulário de avaliação gratuita
2. **Conteúdo** → Relatório personalizado
3. **Autoridade** → Site profissional (ou Instagram forte)
4. **Conversão** → WhatsApp qualificado + follow-up
5. **Gestão** → Dashboard (treinos IA, agenda, financeiro, visão geral)
6. **Retenção** → Acompanhamento (self-efficacy, link semanal, evidência de progresso)
7. **Escala** → Setup de tráfego pago profissional (BM, pixel, segmentação)

**Doc**: `docs/core/funil-proprio/metodo-5-camadas.md`

**How to apply**: usar no relatório diagnóstico, landing, onboarding e toda comunicação.
