Decisoes confirmadas em 2026-04-23 para Fase 4B:

1. **Setup (R$497 site) morre** → servico de criacao customizada de formularios (R$200-300)
2. **Templates pre-built no MVP** (6 templates), sem wizard builder IA no dashboard
3. **Hub publico: modal** de selecao. Link "Analise gratuita" no hub Linktree existente. 1 template = direto, 2+ = modal
4. **Core: 3 templates fixos** (1 por modalidade). Pro: catalogo + edicao limitada
5. **Edicao Pro**: opcoes de resposta + perguntas independentes (sem branches)
6. **Limitar por especialidade**, nao por publico
7. **Foco MVP esportes padrao** — sem gestantes, reabilitacao, terceira idade
8. **Servico Custom (R$200-300)**: manual inicialmente, IA depois

**Why:** fundador identificou que (1) a maioria dos PTs so quer captar leads, nao personalizar templates, (2) wizard builder IA adiciona complexidade/custo sem retorno no MVP, (3) templates especializados precisam de profissionais instruindo.

**How to apply:** Fase 4B implementa hub modal + 6 templates pre-built + dashboard Pro com edicao limitada. Wizard builder IA fica como servico Custom ou feature pos-tracao.
