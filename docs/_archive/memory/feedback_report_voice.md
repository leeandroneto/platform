Report narrative must be written in FIRST PERSON from the professional's perspective, not third person talking ABOUT the professional.

**Why:** Third person ("Leandro vai te mostrar") sounds robotic and detached. First person ("Eu consigo ajustar isso com você") sounds like the professional actually wrote it. The CTA section already has the professional's photo and name — no need to repeat the name 3-5x throughout.

**How to apply:**

- ERRADO: "Com Leandro, dá pra estruturar isso" / "Leandro vai te mostrar"
- CERTO: "Consigo ajustar isso com você" / "Nós vamos alinhar isso" / "Na nossa conversa, a gente monta..."
- Professional name: max 1-2x natural (intro + ponte), not forced repetition
- Tone: as if the professional wrote the text himself
