## Por onde começar

- **Quer criar um template novo?** Leia `MASTER-SPEC.md` na íntegra. Use a §10 (workflow) como guia.
- **Quer ver exemplo de pesquisa pronta?** Abra qualquer `<modalidade>/<NN>/research.md`. Os 33 são exemplos da metodologia aplicada.
- **Quer entender o catálogo de motores/métricas?** §4 (motores) + §8 (métricas) do MASTER-SPEC.
