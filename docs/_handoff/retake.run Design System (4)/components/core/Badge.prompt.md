Compact status/category label — pill by default; use for states (ativo, inadimplente), counts, and tags.

```jsx
<Badge tone="success" dot>Ativo</Badge>
<Badge tone="accent" variant="solid">Novo</Badge>
<Badge tone="neutral" variant="outline">Avançado</Badge>
```

Tones: `neutral | accent | ocean | success | warning | danger`. Variants: `soft` (tinted bg), `solid` (filled), `outline`. Add `dot` for a leading status dot.
